Installation guide for YOURLANGUAGE language pack


1) Download the language pack by clicking on the "Psi Translation - YOURLANGUAGE" link in the upper side of this page.

2) Place the file in the right folder for your operating system:

 - GNU/Linux - Copy psi_xx.qm into $HOME/.local/share/psi/

 - Windows - There are two ways:

   a) Copy psi_xx.qm into C:\Program Files\Psi (if you installed Psi somewhere else, make sure to place it in the right path).

   b) If you don't have administrator privileges in your system, copy psi_xx.qm into %APPDATA%\Psi.

 - Mac OS X - Copy psi_xx.qm into Applications > right-click Psi icon > show package contents > content > resources

3) Close Psi's profile from the main menu, and when selecting it again, choose your language from the list.
